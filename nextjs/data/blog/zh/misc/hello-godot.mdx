按照 Godot 官方文档的步骤，这个项目的目标是用 Godot 引擎做一个简单的 2D 游戏。这个项目主要有两个目的：学习引擎本身，以及理解游戏开发流程。

## 游戏介绍

这是一个简单的 2D 平台游戏。玩家可以使用 `W`、`A`、`S`、`D` 键向左、右、上、下移动。按下 `Enter` 键即可开始游戏。当玩家与敌人发生碰撞时，游戏结束。你可以在[这里](https://jiahaoxiang2000.github.io/hello-godot/)试玩。

## 编码实现

代码实现参考了 Godot 文档中的 [First 2D Game](https://docs.godotengine.org/en/stable/getting_started/first_2d_game/index.html)。游戏使用 GDScript 编写，这是一种用于 Godot 引擎的高级动态类型编程语言。GDScript 与 Python 有些相似，但针对 Godot 做了优化，也提供了诸如语言服务器之类的能力，方便在 VS Code 中开发。

## 游戏部署

一开始，我原本打算把游戏发布到 Windows 平台。不过，Godot 引擎通过 GitHub Actions 进行自动化部署的支持还比较有限。因此，最后把游戏部署到了 GitHub Pages。游戏地址见[这个链接](https://jiahaoxiang2000.github.io/hello-godot/)。
